# بوت إدارة حسابات التليجرام (Telegram Account Manager Bot)

هذا البوت مكتوب بلغة Python ويستخدم مكتبة Pyrogram لإدارة حسابات التليجرام بشكل فردي. يمكن لكل مستخدم إضافة حساباته الخاصة والتحكم بها دون تدخل من المستخدمين الآخرين.

## الميزات

1.  **إدارة الحسابات الفردية:** يمكن لكل مستخدم إضافة حساباته الخاصة باستخدام سلسلة الجلسة (Session String).
2.  **إنشاء المجموعات وإرسال الرسائل:** أمر يدوي لإنشاء مجموعة وإرسال 10 رسائل فيها.
3.  **الجدولة التلقائية لإنشاء المجموعات:** يمكن تشغيل أمر لإنشاء مجموعة جديدة وإرسال 10 رسائل فيها كل 20 دقيقة تلقائياً، مع إمكانية إيقاف التشغيل.

## المتطلبات

*   Python 3.8+
*   مكتبات Python: `pyrogram`, `trio`, `python-dotenv`

## الإعداد والتشغيل

### 1. الحصول على API ID و API Hash

يجب عليك الحصول على `API ID` و `API HASH` من [my.telegram.org](https://my.telegram.org).

### 2. الحصول على Bot Token

قم بإنشاء بوت جديد عبر [BotFather](https://t.me/BotFather) واحصل على `BOT_TOKEN`.

### 3. إعداد ملف `config.env`

قم بإنشاء ملف باسم `config.env` في نفس مجلد المشروع واملأه بالبيانات الخاصة بك:

```env
# بيانات البوت
API_ID=YOUR_API_ID
API_HASH=YOUR_API_HASH
BOT_TOKEN=YOUR_BOT_TOKEN

# معرف المستخدم المالك (لإدارة البوت، يمكن أن يكون معرفك الشخصي)
OWNER_ID=YOUR_OWNER_ID
```

**ملاحظة:** تم تضمين بياناتك التي زودتنا بها في ملف `config.env` بالفعل.

### 4. تثبيت التبعيات

تأكد من أنك داخل البيئة الافتراضية، ثم قم بتثبيت المكتبات المطلوبة:

```bash
pip install -r requirements.txt
```

### 5. تشغيل البوت

قم بتشغيل الملف الرئيسي:

```bash
python main.py
```

## استخدام البوت

1.  **بدء التشغيل:** أرسل `/start` إلى البوت.
2.  **إضافة حساب:** أرسل `/add_account` ثم أرسل سلسلة الجلسة (Session String) الخاصة بحسابك.
3.  **إدارة الحسابات:** أرسل `/my_accounts` لعرض قائمة حساباتك وإدارتها.
    *   من قائمة الإدارة، يمكنك **إنشاء مجموعة وإرسال 10 رسائل** يدوياً.
    *   يمكنك **تشغيل/إيقاف الجدولة التلقائية** لإنشاء مجموعة كل 20 دقيقة.
    *   يمكنك **حذف الحساب** من البوت.

## ملاحظات هامة

*   **سلسلة الجلسة (Session String):** يجب الحصول عليها باستخدام Pyrogram. لا تشاركها مع أي شخص غير موثوق به.
*   **ملف `user_data.json`:** يتم حفظ بيانات الحسابات في هذا الملف. لا تقم بحذفه أو تعديله يدوياً إلا إذا كنت تعرف ما تفعله.
*   **الجدولة التلقائية:** تعمل طالما أن البوت قيد التشغيل. إذا توقف البوت، ستتوقف الجدولة.
